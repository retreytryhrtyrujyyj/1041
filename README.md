# 毕业设计（论文）管理系统

## 项目简介
毕业设计管理系统是学校对毕业论文进行统一管理，分老师和学生两种角色，学生进行选题、中期检查、提交论文等操作，指导老师进行新增选题、选题审核、中期检查、论文评分等操作。对15张数据库表增删改查操作。适合毕业设计、大作业、课程设计。

## 技术介绍
Java;SSM框架；Mysql数据库

## 开发工具
1.IntelliJ IDEA或者Myeclipse 3.Navicat 3.Mysql 4.JDK

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
